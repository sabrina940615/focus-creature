# Focus Creature

一個專注計時應用程式，幫助您在不同的聲音環境中提升專注力。

## 功能

- 🎵 **聲音環境測試** - 測試在不同聲音環境下的專注效果
- ⏱️ **專注計時器** - 自定義時長的專注工作階段
- 🎮 **抑制控制遊戲** - 通過遊戲評估專注力表現
- 📊 **統計分析** - 分析您的專注偏好和趨勢
- 🧚 **精靈收集** - 根據專注表現收集可愛的小精靈

## 如何使用

1. 打開 `index.html` 在瀏覽器中開始使用
2. 完成初始設置
3. 選擇聲音環境進行測試
4. 開始您的專注工作階段

## 技術棧

- 純 HTML / CSS / JavaScript
- IndexedDB 本地數據存儲
- Canvas 圖表渲染
